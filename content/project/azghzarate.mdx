## Features

#### Login Features

- Multiple User Account
- Forgot Password
- 3x attempt with Disabled
- Validations
- User status - active/inactive

#### Inventory Features

- Monitoring Dashboard
- Notification products (Normal / Low on Stocks)
- Sales and Profit (per month)
- Company Details
- Discount
- Supplier Module
- Products Module
- Barcode / Non-Barcode
- Import / Export CSV(Excel)
- Print Products
- Vat / Non-Vatables
- Barcode Generator
- Stock Module
- Sales Report
- More...

#### POS Features

- Start/End of the Day
- User Profile
- New Transaction
- Browse Product / Scan the barcode (automatic)
- More..
